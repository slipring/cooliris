Audio slideshows with Reveal.js

Created by Asvin Goel

Please click the play button in the bottom left corner

The web is full of MOOCs and screencasts based on video recordings of slide decks

  • videos usually are huge files (tons of megabyte)
  • videos are linear and without any possibilities of interaction
  • navigation between slides is hardly possible

With Reveal.js you can easily create much better MOOCs and screencasts

  • per slide audio recording and playback with incredibly small file size
  • interactive elements can be included easily
  • navigation between slides is as simple as a click

Audio playback

To setup audio playback you just have to

  • include the audio-slideshow plugin
  • activate the autoSlide feature
  • provide audio files for each slide and fragment
  • tell the plugin where to find the files

Including the plugin


dependencies: [
	// ...	
	{ src: 'plugin/audio-slideshow/audio-slideshow.js', condition: function( ) { return !!document.body.classList; } },
	// ...	
]

Configuring reveal.js



Reveal.initialize({
	// ...
	autoSlide: 5000,
	audioPrefix: 'audio/',	// audio files are stored in the "audio" folder
	audioSuffix: '.ogg',	// audio files have the ".ogg" ending
	// ...
};

For each slide and fragment the plugin automatically inserts the appropriate audio file and sets the auto slide duration.

Audio recording

To setup audio recording you have to

  • include the slideshow-recorder plugin
  • configure keyboard shortcuts to start/stop recording and to download audio files

Including the plugin


dependencies: [
	// ...	
	{ src: 'plugin/audio-slideshow/slideshow-recorder.js', condition: function( ) { return !!document.body.classList; } },				
	{ src: 'plugin/audio-slideshow/audio-slideshow.js', condition: function( ) { return !!document.body.classList; } },
	// ...	
]

Configuring reveal.js


Reveal.initialize({
	// ...	
	keyboard: { 
		82: function() { Recorder.toggleRecording(); },	// press 'r' to start/stop recording
		90: function() { Recorder.downloadZip(); } 	// press 'z' to download zip containing audio files
	}
	// ...	
};

Recording

  • To start/stop recording just press 'r' (and allow access to the microphone when asked by the browser)
  • A red circle shows that the recorder is on
  • After stopping the recorder, you can click on the play button to check your recording

Navigation while recording

  • When navigating to a slide for which an audio file is already recorded, recording is suspended so that the previously recorded file is not lost
  • A yellow circle shows that recording is automatically resumed when navigating to a slide without a recorded audio file

Preparing an audio slideshow

  • After recording is done, download a zip-file containing all recorded audio files by pressing 'z'
  • All you have to do is to store the audio files in the folder specified for the audio-slideshow plugin

Known issues

  • The recorder has only been tested on Firefox 29 and may not work in other browsers
  • Audio recording in browsers is still under development and the last seconds recorded can be lost (so take a deep breath before switching to the next slide)
